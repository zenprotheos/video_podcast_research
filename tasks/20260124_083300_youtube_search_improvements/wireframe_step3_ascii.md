## Responsive Considerations

### Narrow Viewport (< 800px)
- Summary cards stack vertically (2x2 grid)
- Tabs become dropdown selector
- Table columns become horizontal cards
- Status strip wraps to multiple lines

### Wide Viewport (> 1400px)
- Summary cards remain 4-up
- Tabs remain horizontal with scroll
- Table shows all columns without truncation
- Additional metadata columns visible

## Notes for Implementation

1. **Tab Overflow**: When >5 queries, use dropdown with search filter
2. **Deduplication**: "All Results" shows unique videos; "Query source" column indicates which queries found each video
3. **Pagination**: Independent per tab (All Results vs individual queries)
4. **Real-time Updates**: Summary cards and status strips update as queries complete
5. **Keyboard Navigation**: Tab through checkboxes, arrow keys for table rows
6. **Accessibility**: ARIA labels for all status indicators, screen reader announcements for status changes
